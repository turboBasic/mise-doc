# mise-doc

AI Knowledge Base for [mise](https://github.com/jdx/mise), the polyglot development environment tool.

## Contents

- `cards/` — Knowledge Cards (YAML front-matter + Markdown), one file per topic, organized by category:
  `backends`, `commands`, `concepts`, `configuration`, `environments`, `hooks`, `installation`, `recipes`, `tasks`, `troubleshooting`
- `docs/ai-instructions.md` — source-of-truth instructions for AI agents working in this repo

## Usage

Cards are designed for direct context injection into AI assistants (Claude Code, Copilot, Cursor) and RAG/vector-search pipelines. Loaded by the `mise` Claude Code skill.

Cards are generated by AI agents from upstream sources — see `docs/ai-instructions.md` for the authoritative source list and generation rules.
